AWAK Web login page
To Run Locally
Clone the project

  git clone https://github.com/Zafir258/AWAK_-Web-login-page.git
Go to the project directory

Go to VS code 
Open the terminal 
npm install 

npm init -y
npm install express body-parser


Write command node.js backend/index.js
Open the browser & paste your file path link
